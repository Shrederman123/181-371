ТЕМА ПРОЕКТА

Информационная система для интернет – магазина «Coffee Box»

Общее задание проекта заключается в разработке системы для интернет-магазина: веб-сайт, с использованием системы управления содержимым, включающей в себя систему автоматизированного учета клиентов, продаж и товаров.

Ход работы

Этапы:

  1. Создание каналов коммуникации участников проекта;
  2. Формулировка цели проекта и составление списка задач для её реализации;
  3. Распределение ролей между участниками;
  4. Сбор и анализ необходимой информации;
  5. Разработка технического задания для системы;
  6. Реализация практической части;
  7. Описание процесса работы в итоговом отчете;
  8. Корректировка документации проекта;
  9. Анализ готового проекта;
  10. Подготовка материалов для защиты проекта.
